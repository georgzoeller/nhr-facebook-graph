# nhr-facebook-graph

This module can be used with georgzoeller/node-rest-helper to import helper functions for the Facebook Graph API


## Exported functions

## install
yarn ```install georgzoeller/nhr-facebook-graph```

## build 
(0ptional) ```yarn build```

## test
(Optional) ```yarn test```


## Usage

```

	api = require('node-rest-helper')
	graph = require('nhr-facebook-graph')
	api.load(graph)

```
